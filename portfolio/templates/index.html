Sistema Web de Evaluacion de Empleados

Sistema web, basado en python realizado con el framwork Django, de evaluacion de empleados.
Es un sistema en el cual se asigna una sucursal, se crea un empleado para la sucursal creada y se le asigna un prmedio automaticamente al ingresar valores para las difrenetes aptitudes que el empleado presenta, luego de guardadas en la base de datos se puede ver las evaluaciones realizadas con su respectivo resultado en pantalla
Tambien cuenta con un sistema de ingreso(altas y bajas) de tarjetas de debito a entrgar en cada sucursal, donde se agregan cuando llegan y se eliminan cuando se entregan, este sistema cuenta con una herramienta de busqueda por nombre para facilitar la operativa
